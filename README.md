# Agent Tool Server

Learn how to deploy a custom tool server for AI agents with Ink MCP.

## Tutorial

Full step-by-step tutorial: [docs.ml.ink/examples/ai-and-mcp/agent-tool-server](https://docs.ml.ink/examples/ai-and-mcp/agent-tool-server)

### Prerequisites

- An [Ink](https://ml.ink) account
- Ink MCP connected to your AI agent ([setup guide](https://docs.ml.ink/quick-start))

### Quick Deploy

1. Connect Ink MCP to your agent
2. Prompt: "Build an HTTP API that AI agents can use as a tool server. Include endpoints for web scraping, screenshot capture, and URL shortening. Use Node.js with Express."
3. Prompt: "Deploy with Ink"
